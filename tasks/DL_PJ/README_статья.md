# Система обнаружения сердечных заболеваний на основе ИИ с использованием алгоритмов машинного обучения

**Авторы:** Victor Chang, Vallabhanent Rupa Bhavani, Ariel Qianwen Xu, MA Hossain  
**Журнал:** Healthcare Analytics 2 (2022) 100016  
**DOI:** https://doi.org/10.1016/j.health.2022.100016

---

## Аннотация

Статья посвящена разработке системы обнаружения сердечных заболеваний на основе искусственного интеллекта с применением алгоритмов машинного обучения на языке Python. Основной метод — алгоритм случайного леса (Random Forest), обеспечивающий точность около 83% на обучающей выборке.

---

## Цель исследования

Разработать Python-приложение для предсказания наличия сердечных заболеваний у пациента на основе медицинских данных — с применением методов предобработки данных, разведочного анализа и сравнения нескольких алгоритмов классификации.

---

## Данные

- Источник: набор данных UCI по сердечным заболеваниям (около 300 записей)
- Количество признаков: 14, включая возраст, пол, тип боли в груди, уровень холестерина, давление, ЭКГ-показатели, максимальная частота сердечных сокращений и другие
- Целевая переменная: бинарная (1 — болезнь есть, 0 — нет)

---

## Методология

### Предобработка данных
- Обработка категориальных переменных (One-Hot Encoding)
- Стандартизация числовых признаков (`StandardScaler`)
- Разбиение на обучающую и тестовую выборки

### Разведочный анализ (EDA)
- Построение гистограмм по каждому признаку
- Корреляционная матрица (тепловая карта)
- Анализ распределения целевой переменной

### Используемые алгоритмы
| Алгоритм | Точность |
|---|---|
| K-Nearest Neighbors (KNN) | 87% |
| Random Forest | 84% |
| Support Vector Classifier (SVC) | 83% |
| Decision Tree | 79% |

### Основной алгоритм — Random Forest
Метод ансамблевого обучения, объединяющий множество деревьев решений через механизм голосования:
1. Случайная выборка K точек из обучающего набора
2. Построение дерева решений для каждого подмножества
3. Итоговое предсказание — класс с наибольшим числом голосов

**Точность на обучающей выборке:** ~83%  
**Точность на тестовой выборке:** ~70%

## Ключевые результаты

- Алгоритм случайного леса показал наилучшее сочетание точности и устойчивости среди исследованных методов
- Положительная корреляция выявлена между типом боли в груди и вероятностью заболевания
- Отрицательная корреляция — между максимальной частотой сердечных сокращений и наличием болезни
- Система обеспечивает раннее предсказание сердечных заболеваний по доступным клиническим данным

---

## Ограничения

- Относительно небольшой объём датасета (~300 записей)
- Не учитываются инвазивные методы диагностики (ангиография и др.)
- Точность на тестовой выборке (~70%) ниже, чем на обучающей, что указывает на умеренное переобучение
